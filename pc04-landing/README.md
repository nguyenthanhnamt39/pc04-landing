# PC04 Landing Demo

Landing page demo phong cách glass liquid cho trang chủ Phòng PC04 TP.HCM phục vụ 168 đơn vị Công an cấp xã.

## Thành phần hiện có

- `index.html`: cấu trúc landing page hoàn chỉnh
- `styles.css`: giao diện glass liquid, responsive, civic-tech style
- `app.js`: bộ lọc nhóm đơn vị và tìm kiếm demo phía client

## Cách mở nhanh

- Mở trực tiếp file `index.html` trong trình duyệt
- Hoặc chạy một static server tại thư mục này

## Hướng phát triển tiếp

- Gắn logo thật của đơn vị
- Thay số liệu demo bằng dữ liệu thực tế
- Nối các nút tới cổng đăng nhập, biểu mẫu, danh bạ và thông báo
- Chuyển sang Next.js/Tailwind nếu cần triển khai production quy mô lớn
- Tách dữ liệu đơn vị ra JSON/API để tra cứu thật
